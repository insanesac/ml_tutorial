# ML Coding Interview Checklist

## Can you implement from memory?

### DSA Patterns
- [ ] HashMap (frequency count, duplicate detection)
- [ ] Heap (top-K, kth largest)
- [ ] Sliding Window (fixed + variable)
- [ ] Binary Search (standard + lower/upper bound)
- [ ] Two Pointers (opposite direction)
- [ ] DFS on trees (max depth, count, sum, search)
- [ ] BFS level order
- [ ] Merge intervals

### ML/LLM Coding
- [ ] Cosine similarity
- [ ] Dot product
- [ ] Euclidean distance
- [ ] Top-K retrieval with heap
- [ ] Text chunking (fixed + sliding window)
- [ ] Context window truncation
- [ ] Deduplication (exact + near-duplicate)
- [ ] Precision / Recall / F1
- [ ] Recall@K
- [ ] MRR
- [ ] Reranking

## Before writing code
1. What pattern fits?
2. What data structure?
3. What are the edge cases?
4. What's the complexity?
