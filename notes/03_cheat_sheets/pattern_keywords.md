# Pattern Keywords — Quick Lookup

| Keywords | Pattern |
|---|---|
| frequency, count, duplicate | HashMap |
| unique, seen before | Set |
| top k, ranking, priority | Heap |
| substring, subarray, longest, shortest | Sliding Window |
| sorted array, threshold, first/last | Binary Search |
| overlap, meetings, schedule | Intervals |
| tree, depth, path, hierarchy | DFS |
| level order, shortest path, BFS | BFS |
| pair in sorted array, two sum | Two Pointers |
| embeddings, similarity, semantic search | Cosine Similarity + Heap |
| context window, token limit | Greedy Reverse |
| chunk, split documents | Sliding Window |
| deduplication | Set (exact) / Similarity (near) |
| ranking quality | Recall@K, MRR, NDCG |
| RAG, retrieval, knowledge base | Embed → Similarity → Top K → Rerank |
